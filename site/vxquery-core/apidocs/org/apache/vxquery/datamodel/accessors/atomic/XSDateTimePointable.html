    edu.uci.ics.hyracks.data.std.api.IPointable, edu.uci.ics.hyracks.data.std.api.IValueReference, IDate, ITime, ITimezone

    public class XSDateTimePointable
extends edu.uci.ics.hyracks.data.std.api.AbstractPointable
implements IDate, ITime, ITimezone
    The datetime is split up into eight sections. Due to leap year, we have decided to keep the storage simple by saving each datetime section separately. For calculations you can access YearMonth (months) and DayTime (milliseconds) values.
